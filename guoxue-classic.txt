### 1. 首页设计提示词
设计一个国学学习网站首页，包含以下元素：
1. 中国风水墨元素作为背景，顶部为简洁导航栏
2. 醒目标语"传承国学经典，弘扬中华文化"配以书法字体
3. 设计四个精选经典书籍展示卡片（四书或五经），每张卡片包含书籍封面、名称和简短介绍
4. 底部设计三个功能区块：今日名句、学习路径、线上讲堂
5. 页面整体风格简约大方，色调以墨黑、赭石红、米白为主
6. 确保设计响应式，适配手机和电脑

### 2. 经典著作库页面提示词
设计国学经典著作库页面，要求：
1. 顶部设计分类筛选器：按朝代、按流派（儒家、道家、法家等）、按形式（诗词、散文等）
2. 设计书籍列表展示，每本书包含：封面图、书名、作者、朝代、简介
3. 左侧设计树状分类导航：四书五经、诸子百家、历史典籍、诗词歌赋等
4. 添加搜索框，支持按书名、作者、内容关键词搜索
5. 分页设计，每页显示12本书籍
6. 鼠标悬停时显示"开始阅读"和"收藏"按钮

### 3. 书籍详情页提示词
设计国学经典书籍详情页，以《论语》为例：
1. 顶部展示书籍基本信息：书名、作者、朝代、历史地位
2. 设计三栏式阅读区：原文、注释、译文并列展示
3. 章节目录侧边栏，可折叠
4. 阅读功能区：字体大小调节、白天/夜间模式、书签、笔记
5. 页面右侧设计工具栏：查词解字、复制、分享
6. 底部设计相关推荐和读者评论区
7. 加入音频朗读功能按钮

### 4. 国学名句页面提示词
设计国学名句赏析页面：
1. 顶部为精选名句展示区，采用书法字体展示，配以简约山水背景
2. 每日一句模块，包含原文、出处、译文和现代应用
3. 设计名句分类标签云：修身、齐家、治国、平天下、人生哲理等
4. 名句卡片设计，包含：原文、出处、解读、点赞和分享功能
5. 提供"名句接龙"互动游戏模块
6. 底部设计用户投稿入口和热门名句排行榜

### 5. 学习社区页面提示词
设计国学学习社区页面：
1. 顶部用户个人学习数据概览：学习天数、阅读书籍数、笔记数
2. 设计学习笔记发布编辑器，支持富文本和图片上传
3. 笔记流展示区，按热度或时间排序
4. 学习小组模块，展示各类主题学习小组
5. 设计问答区，用户可提问关于国学的疑问
6. 侧边栏展示活跃用户和优质创作者
7. 设计线上读书会活动日历和报名入口

### 6. 个人学习中心提示词
设计用户个人学习中心页面：
1. 学习数据仪表盘：累计学习时间、阅读进度、完成作品数
2. 学习日历热力图，展示每日学习情况
3. 个人收藏书架，展示已收藏的经典著作
4. 学习笔记管理区，分类展示个人笔记
5. 学习计划制定工具，支持设定目标和提醒
6. 个人成就徽章展示区
7. 学习记录时间线，可回顾过去学习内容

### 7. 国学讲堂提示词
设计国学讲堂页面：
1. 顶部轮播展示精品课程或名师讲座
2. 课程分类导航：经典导读、国学与现代生活、传统文化精髓等
3. 课程卡片设计，包含：课程封面、标题、讲师、时长、评分
4. 视频播放页面设计，包含笔记工具和问答区
5. 讲师介绍区，展示讲师背景和其他课程
6. 设计学习路径推荐，从入门到精通的课程系列
7. 添加课程日历，展示即将开始的直播课程

### 8. 移动端适配提示词
为国学学习网站设计移动端界面：
1. 设计自适应导航菜单，在手机上转为底部导航栏
2. 首页内容单列排布，确保阅读舒适度
3. 阅读界面优化，支持上下滑动和左右翻页
4. 设计触屏友好的交互元素，按钮尺寸合理
5. 提供"摇一摇"随机名句功能
6. 优化登录注册流程，支持手机号快速登录
7. 设计离线阅读功能，支持内容缓存

### 9. 国学知识测验提示词
设计国学知识测验功能：
1. 测验分类：经典常识、名句填空、人物典故、文化常识
2. 设计趣味答题界面，融入中国传统元素
3. 每道题提供详细解析和原文引用
4. 设计积分奖励系统和排行榜
5. 提供"每日一题"推送功能
6. 设计进阶挑战模式，难度递增
7. 测验结果分析页面，展示知识掌握情况

### 10. 传统节日专题提示词
设计传统节日专题页面模板：
1. 节日主题banner设计，融入节日特色元素
2. 节日介绍区，包含起源、习俗、文化内涵
3. 相关经典文学作品推荐（如端午与屈原作品）
4. 设计节日习俗互动展示
5. 古今对比区，展示节日的历史变迁
6. 节日相关成语故事和诗词
7. 用户分享区，可上传节日庆祝照片和心得

### 11. 注册登录流程提示词
设计国学网站用户注册登录流程：
1. 简洁登录界面，支持手机号、邮箱和第三方账号登录
2. 注册页设计，引导用户填写基本信息和学习兴趣
3. 设计新用户引导流程，推荐初始学习内容
4. 个性化头像选择，提供传统文化元素头像
5. 用户角色选择：学生、教师、文化爱好者等
6. 隐私设置选项，允许用户控制数据分享范围
7. 设计忘记密码和账号找回流程

### 12. 技术实现提示词
国学学习网站技术架构设计：
1. 前端使用Vue.js框架，实现SPA应用
2. 响应式设计，采用移动优先策略
3. 后端采用Node.js，提供RESTful API
4. 数据库设计：用户信息、书籍内容、学习记录、社区内容
5. 全文检索实现，支持模糊匹配和多维度搜索
6. 用户学习数据分析系统，提供个性化推荐
7. CDN加速和缓存策略，优化图片和视频加载
8. 安全措施：数据加密、XSS防御、CSRF保护
